OSCAR with Wikidata

This interface let you perform queries on the Wikidata triplestore. It represent an alternative configuration and usage of OSCAR: an RDF search application. We have decided to limit the range of data to be searched to bibliographic entities within the scholarly domain. You can try to type: DOIs, ORCIDs and free textual inputs. Here we list some examples:

  • DOI: 10.1186/1471-2105-10-S6-S19
  • ORCID: 0000-0003-0530-4305
  • Document titles with keyword: "semantic"

NOTE: Searches might take several seconds, please be patient.
