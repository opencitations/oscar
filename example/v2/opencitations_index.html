OSCAR with COCI

This interface let you perform queries on the COCI triplestore. It represent an alternative configuration and usage of OSCAR: an RDF search application. Here we list some examples:

  • OCI Identifier: 02001010806360107050663080702026306630509-02001010806360107050663080702026305630301
  • Citations of a specific DOI: 10.1186/1756-8722-5-31
  • References of a specific DOI: 10.1186/1756-8722-6-59

NOTE: Searches might take several seconds, please be patient.
